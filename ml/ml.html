课程简介

人工智能是计算机科学中涉及研究、设计和应用智能机器的一个分支。本课程是计算机科学与技术，以及有关专业重要的专业方向与特色模块课程之一。通过本课程学习，使学生对人工智能的发展概况、基本原理和应用领域有初步了解，对主要技术及应用有一定掌握，启发学生对人工智能的兴趣，培养知识创新和技术创新能力。通过学习，学生能够知道什么时候需要某种合适的人工智能方法用于给定的问题，并能够选择适当的实现方法。

本课程对学生的要求：对本课程的预修课程高等数学、概率论、矩阵论进行复习，了解离散数学课程中的相关内容，掌握一名高级程序设计语言。课前预习，课后完成所布置的作业和程序设计，并了解与本课程相关的本专业前沿知识。

本课程考核分阶段分层次进行，考查学生课堂学习、课外作业、动手实践及自主研究学习的综合能力，考核内容、形式及权重见表。

课程实验

本实验是为了配合《人工智能及应用》课程的理论学习而专门设置的。本实验的目的是巩固和加强人工智能的基本原理和方法，并为今后进一步学习更高级课程和信息智能化技术的研究与系统开发奠定良好的基础。 全书共分为六个实验：1.A*算法求解8数码问题；2. 利用α-β搜索过程的博弈树搜索算法编写一字棋游戏；3.线性分类器的设计实现；4.感知器算法的设计实现；5.SVM线性分类器的设计实现；6.卷积神经网络CNN框架的设计实现。

每个实验均需独立完成，实验报告独立成册，包括封页及所有实验报告内容，实验报告具体要求详见附件。同时，每次实验均需提交实验相关代码。所有实验资料放入一个文件夹，文件夹命名方式为：学号_姓名_实验X。

本课程实验主要使用MatLab完成。MATLAB 学习资源参考： MATLAB documentation pages.

课程设计

Your class project is an opportunity for you to explore an interesting machine learning problem of your choice in the context of a real-world data set. Below, you will find some project ideas, but the best idea would be to combine machine learning with problems in your own research area. Your class project must be about new things you have done this semester; you cannot use results you have developed in previous semesters.

Projects can be done in teams of 4~5 students. For a group, group members are responsible for dividing up the work equally and making sure that each member contributes. Your project will have 4 deliverables:

  • Proposal: 1~2 page [Due Date:]
  • Presentation: About 5-8 minutes per group
    • Date:
    • Time:
    • Venue:
  • Final Report: 6~8 pages [Due Date:]

For the project proposal, each team needs to prepare a one-page report stating the problem, who will participate, and what the expected outcomes will be. In the final report, you need an abstract, introduction, a discussion of related work, a description of the main idea, a description of the data, algorithms, experiments and a conclusion. Also, source code and data set should be attached, such that the work is sufficiently reproducible.

Consider using sample project topics from the following sources:

  • Carlos Guestrin's class at CMU.
  • Goeff Gordon's class at CMU
  • Ray Mooney's class, UT Austin
  • Andreas Krause's class, Caltech
  • Tom Mitchell's class, CMU
  • Apply machine learning algorithms in novel ways to datasets from UCI machine learning repository
Sample project reports are as follows Project reports from Andrew Ng's class 2011
